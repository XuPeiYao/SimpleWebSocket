SimpleWebSocket
=====
[![xpy MyGet Build Status](https://www.myget.org/BuildSource/Badge/xpy?identifier=58572f41-bfc9-4394-b976-e7cc1275e6be)](https://www.myget.org/)

### Install Packages(安裝)
1. [MyGet](https://www.myget.org/feed/xpy/package/nuget/SimpleWebSocket)
